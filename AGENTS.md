
# OpenWave – Codex Agent Guidelines

This repository contains the OpenWave AI news briefing application.

Project structure must remain stable.

Repository structure:

backend/
flutter_app/
docs/

Do NOT create alternative folders such as:
frontend/
mobile/
app/

When implementing tasks:

1. Modify only the necessary files.
2. Do not change repository structure unless explicitly requested.
3. Commit changes after completing a task.
4. Show the list of modified files.

Backend:
- FastAPI application
- endpoints: /articles, /briefing/today

Mobile:
- Flutter application
- main screens: Home, Player

Documentation must be updated when architecture changes:
docs/PROJECT_CONTEXT.md
docs/TASKS.md
docs/DAILY_LOG.md
docs/ARCHITECTURE.md
