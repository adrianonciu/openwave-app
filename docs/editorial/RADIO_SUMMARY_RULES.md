# Reguli Pentru Rezumatul Radio

## Obiectiv
Fiecare stire din OpenWave trebuie sa poata fi ascultata rapid si inteleasa fara suport vizual. Rezumatul radio este construit pentru ureche, nu pentru lectura pe ecran.

## Lungime standard
- Fiecare rezumat are intre 3 si 5 propozitii.
- Prima propozitie livreaza esenta evenimentului.
- Urmatoarele propozitii explica impactul, contextul si ce urmeaza.

## Structura recomandata
1. Ce s-a intamplat.
2. Cine este afectat sau de ce conteaza.
3. Contextul minim necesar.
4. Urmatorul pas, daca este relevant.

## Reguli de formulare
- O idee principala per propozitie.
- Propozitii scurte, respirabile si usor de citit cu voce tare.
- Se prefera ordinea subiect, verb, complement.
- Se evita parantezele, abrevierile obscure si listele lungi in interiorul frazei.
- Se evita cifrele excesive; se pastreaza doar cele esentiale.

## Citate
- Citatele sunt optionale, nu obligatorii.
- Un citat se foloseste doar daca adauga claritate, tensiune jurnalistica sau o formulare memorabila.
- Citatul trebuie sa fie scurt si usor de rostit.

## Reguli privind sursele
- Rezumatul sintetizeaza informatii din 2 pana la 4 surse.
- Referintele surselor apar numai in versiunea scrisa a briefingului.
- Sursele nu sunt rostite in audio.

## Ce se evita
- Introduceri teatrale sau dramatizare artificiala.
- Fraze lungi cu multe subordonate.
- Ambiguitati de tipul "situatia ramane in atentie" fara explicatie concreta.
- Limbaj vag precum "lucrurile evolueaza" fara detaliu util.
