# Modelul De Flux Radio

## Scop
Modelul de flux radio defineste ordinea ideilor intr-o stire astfel incat ascultatorul sa primeasca rapid esentialul, apoi contextul si, la final, perspectiva utila.

## Flux standard
1. Deschidere: evenimentul principal.
2. Impact: cine este afectat si de ce conteaza.
3. Context: ce trebuie stiut pentru intelegere.
4. Evolutie: ce urmeaza sau care este starea actuala.

## Principii de compozitie
- Se intra direct in subiect.
- Se prioritizeaza utilitatea pentru public.
- Se evita detaliile secundare inaintea ideii principale.
- Se inchide cu o informatie orientata spre urmatorul pas.

## Variatii permise
- Pentru breaking news, impactul poate veni imediat dupa deschidere.
- Pentru subiecte explicative, contextul poate fi extins intr-o a patra sau a cincea propozitie.
- Pentru subiecte controversate, se adauga o mentiune scurta privind lipsa consensului.

## Relatia cu sursele
- Materialul final rezulta din sinteza a 2 pana la 4 surse.
- In briefingul audio nu se citesc sursele.
- In produsul scris, sursele sunt listate la finalul briefingului.
