# OpenWave Transitions RO

## Scop
Acest document defineste formule de tranzitie radio in limba romana pe care generarea OpenWave le poate folosi intre subiecte. Tranzitiile trebuie sa ajute curgerea jurnalului fara sa incarce ritmul.

## Regula generala
- Tranzitiile se folosesc rar si cu masura.
- O regula practica este aproximativ o tranzitie la fiecare 2 sau 3 stiri.
- Nu fiecare item are nevoie de o punte explicita.
- Daca ordinea jurnalului este deja clara, tranzitia poate lipsi.

## Tranzitii neutre
- "Tot astazi..."
- "Intre timp..."
- "In acelasi timp..."
- "Mai este ceva important..."
- "Tot in aceasta zi..."

## Tranzitii de contrast
- "Pe de alta parte..."
- "In schimb..."
- "Dar in alta parte..."
- "Totusi..."

## Tranzitii de continuare
- "In continuare..."
- "Mai departe..."
- "Tot din acest tablou..."
- "Ramanem in aceeasi tema..."

## Tranzitii spre utilitate publica
- "Pentru public, miza este simpla..."
- "Pentru cei care urmaresc subiectul, conteaza ca..."
- "Dincolo de declaratii, important este ca..."

## Tranzitii spre final
- "Iar la final..."
- "In incheiere..."
- "Ultima oprire din jurnal vine din..."

## Reguli de folosire
- Tranzitia trebuie sa potriveasca relatia reala dintre povesti: continuitate, contrast sau schimbare de registru.
- Nu se folosesc mai multe tranzitii puternice una dupa alta.
- "Pe de alta parte..." se foloseste doar cand chiar exista contrast.
- Tranzitia nu trebuie sa repete titlul sau ideea principala deja spusa.
- Daca o punte suna prea scris sau prea teatrala, ea trebuie eliminata.
