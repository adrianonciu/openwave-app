# Roadmap OpenWave

## MVP
- ingestie RSS de baza
- clustering simplu al evenimentelor
- generare de briefing zilnic
- ecran principal si player audio
- reguli editoriale de baza pentru rezumate radio

## Beta
- personalizare pe domenii si geografie
- scorare mai buna a importantei
- validare editoriala pentru controverse si credibilitate
- lista scrisa a surselor la finalul briefingului

## Version 1
- briefinguri mai stabile si mai bine ordonate
- extinderea taxonomiei de domenii
- preferinte pentru publicatii si orientare editoriala
- optimizare a experientei audio si a retentiei in player

## Version 2
- personalizare avansata pe comportament si context de consum
- explicatii mai bune pentru subiecte complexe
- suport extins pentru editii locale si internationale
- imbunatatire continua a redactiei automatizate pe baza feedbackului
